# [initial] section

Initial post-installation settings

## admin_user_name

Initial NOC admin's username

|                |                               |
| -------------- | ----------------------------- |
| Default value  | `admin`                       |
| YAML Path      | `initial.admin_user_name`     |
| Key-Value Path | `initial/admin_user_name`     |
| Environment    | `NOC_INITIAL_ADMIN_USER_NAME` |
