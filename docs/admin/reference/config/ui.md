# [ui] section

[ui](../services/ui.md) service configuration

## max_avatar_size

Maximum allowed avatar image size, in bytes.

|                |                          |
| -------------- | ------------------------ |
| Default value  | `256K`                   |
| YAML Path      | `ui.max_avatar_size`     |
| Key-Value Path | `ui/max_avatar_size`     |
| Environment    | `NOC_UI_MAX_AVATAR_SIZE` |
