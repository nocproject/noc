# [home] section

Home screen configuration

## enable_welcome

Enable "Welcome to NOC" widget.

|                |                           |
| -------------- | ------------------------- |
| Default value  | `True`                    |
| YAML Path      | `home.enable_welcome`     |
| Key-Value Path | `home/enable_welcome`     |
| Environment    | `NOC_HOME_ENABLE_WELCOME` |
