[runner](../services-reference/runner.md) service configuration

## max_running

Maximal amount of jobs which can be in `Running` state.

Default value
: `10`

=== "YAML"

    `runner.max_running`

=== "Key-value"

    `runner/max_running`

=== "Environment"

    `NOC_RUNNER_MAX_RUNNING`
